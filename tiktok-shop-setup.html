まず4Sellerの無料アカウントを作成します

4Sellerは無料で使えるマルチチャネル管理ツールです。Amazon と TikTok Shop を繋ぐミドルウェアとして機能します。ELEVAとの連携に必要なため、まず4Sellerに登録してください。

4Seller
https://www.4seller.com
4Sellerを開く →
ELEVAとの連携確認用に保存されます。
4Sellerに登録する →

4SellerとAmazon Seller Centralを接続します

4Sellerのダッシュボードから Amazon出品アカウントを連携させてください。これにより FBA 在庫の自動管理が可能になります。

  1. 4Sellerにログイン →「Channels」をクリック
  2. 「+ Add Channel」→「Amazon」を選択
  3. 「Connect with Amazon」をクリック
  4. Amazon Seller Centralのログイン画面でログイン
  5. アクセス許可を承認
PDFガイドを見る

4SellerとTikTok Shopを接続します

TikTok Shopのセラーアカウントと4Sellerを連携させます。これにより、TikTok Shopの注文が自動で Amazon FBA から発送されます。

  1. 4Sellerにログイン →「Channels」をクリック
  2. 「+ Add Channel」→「TikTok Shop」を選択
  3. TikTokアカウントでログインして承認
  4. 「Product Sync」を ON にする
  5. 「Order Routing」を ON にする（MCF自動出荷）
PDFガイドを見る

販売する商品をELEVAに登録します

動画を作りたい商品を登録してください。損益計算と動画生成に使用します。
