結果パターン別の対処

  • A: TEST 1 のレスポンスに gemini-2.5-flash が無い → Generative Language API が新プロジェクトで未有効化。
    https://console.cloud.google.com/apis/library/generativelanguage.googleapis.com を開き、Gemini Project（APIA ではない）を選択して「有効にする」。
  • B: TEST 1 OK / TEST 2 が 403 → API キーに API restrictions がかかっている。Google AI Studio で「Don't restrict key」に変更。
  • C: TEST 2 OK / TEST 3 が 403 → image モデルがリージョン制限されている。gemini-3-flash-preview または imagen-4-fast へのフォールバックを検討。
  • D: 全部 OK → analyze-buzz-video 側のモデル名 / プロンプトの問題。生成パラメータを見直し。
